OLLIM 사용 가이드

성공적인 투자를 위한 첫걸음

📱 스마트폰 앱처럼 설치하기

  1. Safari 브라우저로 접속
  2. 하단 중앙 공유 버튼(⬆️) 클릭
  3. [홈 화면에 추가] 선택
  4. 우측 상단 [추가] 클릭
  1. Chrome 또는 삼성인터넷 접속
  2. 우측 상단 혹은 하단 메뉴(⋮ 또는 ☰) 클릭
  3. [앱 설치] 또는 [홈 화면에 추가] 선택

💡 홈 화면 아이콘으로 접속 시 주소창 없이 풀스크린으로 이용 가능합니다.

🔐 보안 및 데이터 관리

  • 비밀번호: 최초 접속 시 구매 안내받은 비번을 입력하세요.
  • 저장 방식: 데이터는 서버가 아닌 기기 내부(로컬)에만 저장됩니다. 브라우저 캐시 삭제 시 데이터가 소실될 수 있습니다.
  • 백업: 기기 변경 시 [📥 내보내기]로 CSV를 저장한 후, 새 기기에서 [📤 불러오기] 하세요.

📊 주요 기능 활용

  • 스마트 히스토리: [Load More]로 과거 내역을 확인하고, [Close Records]로 화면을 깔끔하게 접으세요.
  • 실시간 대시보드: 입력 즉시 자산 곡선과 수익률이 업데이트되며, 목표 달성 등급(Lv)이 표시됩니다.

⚠️ 저작권 고지: 본 프로그램의 저작권은 OLLIM에 있으며, 무단 복제 및 배포 시 법적 처벌을 받을 수 있습니다.

🛠️ 초기화: 데이터 오류 시 [Factory Reset]을 사용하세요. (복구 불가)
